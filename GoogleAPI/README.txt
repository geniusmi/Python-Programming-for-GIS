This repository shows how to use Google API service and how to handle JSON data on Python 2.7.
To use Google API services, you need to register an account to get your Google API key at https://developers.google.com/maps/documentation/android-sdk/get-api-key

getElevation.py can get the the elevation data of given latitude and longitude in JSON format.

AddressToCoordinates.py gets the coordinates of given addressses.
